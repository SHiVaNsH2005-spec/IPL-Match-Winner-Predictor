# 🏏 IPL Match Winner Predictor

A Machine Learning project that predicts the winner of an IPL match using historical IPL data and Random Forest Classification.

## 🚀 Features

- IPL Match Winner Prediction
- Winning Probability Score
- Head-to-Head Statistics
- Team & Venue Selection
- Toss Impact Analysis
- Interactive Gradio Dashboard
- Data Cleaning and Preprocessing

## 📊 Dataset

The project uses historical IPL match data containing:

- Team 1
- Team 2
- Venue
- Toss Winner
- Toss Decision
- Match Winner

## 🧠 Machine Learning Model

Algorithm Used:

- Random Forest Classifier

## 🛠️ Technologies Used

- Python
- Pandas
- Scikit-Learn
- Gradio
- Google Colab

## 📈 Model Workflow

1. Data Collection
2. Data Cleaning
3. Feature Selection
4. Label Encoding
5. Train-Test Split
6. Random Forest Training
7. Prediction
8. Interactive Deployment using Gradio

## ▶️ How to Run

```bash
pip install pandas scikit-learn gradio
```

Run the notebook or Python script and launch the Gradio interface.

## 📷 Project Preview

Interactive dashboard for predicting IPL match winners based on teams, venue, toss winner and toss decision.
